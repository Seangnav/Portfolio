JavaFX Blackjack Game

A desktop application that recreates the classic casino card game using Java and JavaFX. It features an interactive GUI with card graphics, player vs dealer gameplay, and full Blackjack rules (hit, stand, bust, dealer AI). The project highlights object-oriented design, event-driven programming, and GUI development in Java.
